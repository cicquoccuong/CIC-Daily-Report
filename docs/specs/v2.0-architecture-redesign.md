## 2. WHAT: Thay \u0111\u1ed5i g\u00ec

### 2.1 Ki\u1ebfn tr\u00fac Daily Pipeline m\u1edbi: Master Analysis -> Tier Extraction

**Hi\u1ec7n t\u1ea1i (v0.32.0)**:
```
Data (kh\u00e1c nhau m\u1ed7i tier) -> 6 LLM calls song song -> 6 b\u00e0i ri\u00eang bi\u1ec7t
```

**M\u1edbi (v2.0)**:
```
ALL Data + Expert Consensus + Historical
    -> 1 Master Analysis (~5000 t\u1eeb, to\u00e0n di\u1ec7n)
    -> Quality Gate (ki\u1ec3m tra factual consistency)
    -> 5 Tier Extractions (L1-L5) + Summary + Research
```

**Summary extraction**: S\u1eed d\u1ee5ng story-based digest format \u2014 m\u1edf \u0111\u1ea7u b\u1eb1ng 1 \u0111o\u1ea1n "C\u1eadp nh\u1eadt Th\u1ecb tr\u01b0\u1eddng" (s\u1ed1 li\u1ec7u l\u1ed3ng v\u00e0o c\u00e2u v\u0103n, KH\u00d4NG b\u1ea3ng emoji), ti\u1ebfp theo l\u00e0 t\u1eebng s\u1ef1 ki\u1ec7n ri\u00eang bi\u1ec7t (ti\u00eau \u0111\u1ec1 \u0111\u1eadm + 1-2 \u0111o\u1ea1n ph\u00e2n t\u00edch s\u00e2u). Expert consensus section \u0111\u01b0\u1ee3c th\u00eam ngay sau market overview.

**Chi ti\u1ebft story-based digest format (v0.31.0, \u0111\u00e3 tri\u1ec3n khai trong `summary_generator.py`)**:

1. **Hook** (1-2 c\u00e2u): Cross-signal divergence ho\u1eb7c s\u1ed1 li\u1ec7u b\u1ea5t ng\u1edd t\u1eeb data
2. **C\u1eadp nh\u1eadt Th\u1ecb tr\u01b0\u1eddng** (1 \u0111o\u1ea1n): S\u1ed1 li\u1ec7u l\u1ed3ng v\u00e0o v\u0103n xu\u00f4i nh\u00e2n qu\u1ea3
3. **5-8 tin t\u1ee9c** s\u1eafp theo m\u1ee9c quan tr\u1ecdng: TOP 2-3 vi\u1ebft s\u00e2u (2 \u0111o\u1ea1n), ph\u1ee5 3-5 vi\u1ebft ng\u1eafn (1 \u0111o\u1ea1n)
4. **\ud83d\udcc5 S\u1eafp t\u1edbi** (conditional): Macro/crypto events 3-7 ng\u00e0y t\u1edbi

**Rules**:
- Max 3 c\u00e2u/\u0111o\u1ea1n
- Bold ch\u1ec9 cho ti\u00eau \u0111\u1ec1 + s\u1ed1 then ch\u1ed1t
- Telegram copy-paste ready
- Kh\u00f4ng d\u00f9ng b\u1ea3ng/emoji table

**L\u00fd do**:
- Master Analysis nh\u00ecn th\u1ea5y T\u1ea4T C\u1ea2 data -> ph\u00e2n t\u00edch s\u00e2u nh\u1ea5t
- 1 b\u00e0i g\u1ed1c -> kh\u00f4ng th\u1ec3 m\u00e2u thu\u1eabn gi\u1eefa tier
- Token t\u1ed5ng: 1x15K (Master) + 7 extractions (L1-L5 + Summary + Research, trung b\u00ecnh ~3K m\u1ed7i c\u00e1i) = ~36K output tokens. Cao h\u01a1n hi\u1ec7n t\u1ea1i (~25K) nh\u01b0ng ki\u1ebfn tr\u00fac single-source lo\u1ea1i b\u1ecf m\u00e2u thu\u1eabn v\u00e0 cho ph\u00e9p ph\u00e2n t\u00edch s\u00e2u h\u01a1n m\u1ed7i token.

### 2.2 Expert Consensus Engine (MODULE M\u1edaI)

6 ngu\u1ed3n, chia 3 layers:

**Layer 1: Quantitative (structured data, kh\u00f4ng c\u1ea7n NLP)**
- Polymarket Gamma API (free, no auth) \u2014 implied probabilities
- Kalshi API (free, no auth for read) \u2014 crypto markets (Phase 2 fallback)
- CryptoPanic sentiment (\u0111\u00e3 c\u00f3) \u2014 bullish/bearish ratio
- Augmento Bull/Bear Index (free, delayed) \u2014 social sentiment 0-1
- Alternative.me F&G (\u0111\u00e3 c\u00f3)

**Layer 2: Expert Opinion (NLP via LLM classification)**
- 39 Telegram channels (Telethon, \u0111\u00e3 c\u00f3 code placeholder) \u2014 classify m\u1ed7i message: Bullish/Neutral/Bearish + key levels + thesis
- TradingView popular ideas (tradingview-scraper library) \u2014 sentiment + key levels + target
- YouTube transcripts (youtube-transcript-api, free, no key) \u2014 sentiment + main argument
- RSS research reports NLP layer (Glassnode Insights, Messari) (Phase 3)

**Layer 3: Smart Money Actions (revealed preference)**
- Whale Alert (\u0111\u00e3 c\u00f3)
- CryptoQuant alerts + Arkham Intelligence alerts (via TG channels, included in 39-channel list)
- ETF flows (\u0111\u00e3 c\u00f3 trong research_data.py)

**Consensus Scoring Algorithm**:
```python
# Weights
WEIGHTS = {
    "prediction_markets": 3.0,  # skin in the game
    "smart_money": 2.5,         # actions > words
    "expert_channels": 2.0,     # track record
    "research_reports": 2.0,    # institutional
    "tradingview": 1.5,         # community traders
    "social_sentiment": 1.0,    # retail
}

# Output per asset (BTC, ETH, market_overall)
@dataclass
class MarketConsensus:
    asset: str
    score: float  # -1.0 to +1.0
    label: str  # "STRONG_BULLISH" / "BULLISH" / "NEUTRAL" / "BEARISH" / "STRONG_BEARISH"
    sources: list[ConsensusSource]
    key_levels: KeyLevels  # support, resistance from TradingView
    contrarians: list[ConsensusSource]  # sources disagreeing with consensus
    divergence_alerts: list[str]  # when smart money disagrees with experts
    polymarket_probability: dict  # e.g., {"BTC>70K_apr": 0.72}
```
